# Flash-Study:

Flash-study is a mobile friendly study aid web application. A user can create flashcards for different subjects, study the flashcards, then test on those subjects.

## Technologies:

* React
* Redux
* JavaScript(ES6)
* NodeJS
* ExpressJS
* Massive
* PostgreSQL
* Axios
* Chart.js
* Auth0
